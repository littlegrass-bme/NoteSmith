# 课程笔记格式规范详解

## Markdown 格式规范

### 一、页面布局与边距

#### 页边距（PDF输出）
- **上边距**：2.5 cm
- **下边距**：2.5 cm
- **左边距**：2.5 cm
- **右边距**：2.5 cm
- **行间距**：1.5倍

#### 页面设置
- **纸张大小**：A4（210 × 297 mm）
- **页面方向**：纵向（Portrait）
- **页脚**：包含页码（右对齐）
- **页眉**：可选，通常为课程名称或章节名

### 二、字体与字号

#### 正文
- **字体**：宋体 或 Calibri
- **字号**：12 pt
- **颜色**：黑色（#000000）
- **行距**：1.5倍

#### 标题层级

##### 一级标题（# 标题）
- **字体**：黑体 或 Arial Bold
- **字号**：18 pt
- **颜色**：黑色
- **加粗**：是
- **上边距**：24 pt
- **下边距**：12 pt
- **对齐**：左对齐
- **前面加**：无
- **后面加**：无

**Markdown示例：**
```markdown
# 课程笔记标题
```

**效果：**
# 课程笔记标题

---

##### 二级标题（## 标题）
- **字体**：黑体 或 Arial Bold
- **字号**：16 pt
- **颜色**：黑色
- **加粗**：是
- **上边距**：18 pt
- **下边距**：10 pt
- **对齐**：左对齐
- **前面加**：无
- **后面加**：无

**Markdown示例：**
```markdown
## 主要章节标题
```

**效果：**
## 主要章节标题

---

##### 三级标题（### 标题）
- **字体**：黑体 或 Arial
- **字号**：14 pt
- **颜色**：黑色
- **加粗**：是
- **上边距**：14 pt
- **下边距**：8 pt
- **对齐**：左对齐
- **前面加**：无
- **后面加**：无

**Markdown示例：**
```markdown
### 小节标题
```

**效果：**
### 小节标题

---

##### 四级标题（#### 标题）
- **字体**：黑体 或 Arial
- **字号**：12 pt
- **颜色**：黑色
- **加粗**：是
- **上边距**：10 pt
- **下边距**：6 pt
- **对齐**：左对齐

**Markdown示例：**
```markdown
#### 子小节标题
```

**效果：**
#### 子小节标题

---

### 三、段落格式

#### 段落缩进
- **首行缩进**：2字符（约0.5 cm）
- **段前间距**：0 pt
- **段后间距**：6 pt

**Markdown实现方式：**
```markdown
　　这是一个缩进的段落。使用两个全角空格实现首行缩进。

　　这是第二个段落，也有缩进。
```

#### 段落对齐
- **正文段落**：两端对齐（Justify）
- **标题**：左对齐（Left）
- **特殊内容**：根据内容类型调整

#### 段落间距
- **同级段落之间**：6 pt
- **不同级标题之间**：12 pt
- **段落与列表之间**：6 pt

### 四、列表格式

#### 无序列表（项目符号）
- **符号类型**：圆点（•）或短横线（-）
- **缩进**：0.5 cm
- **行间距**：单倍行距
- **项目间距**：3 pt

**Markdown示例：**
```markdown
- 第一项
- 第二项
- 第三项
```

**效果：**
- 第一项
- 第二项
- 第三项

#### 有序列表（编号）
- **编号类型**：数字（1. 2. 3.）
- **缩进**：0.5 cm
- **行间距**：单倍行距
- **项目间距**：3 pt

**Markdown示例：**
```markdown
1. 第一项
2. 第二项
3. 第三项
```

**效果：**
1. 第一项
2. 第二项
3. 第三项

#### 嵌套列表
- **一级缩进**：0.5 cm
- **二级缩进**：1.0 cm
- **三级缩进**：1.5 cm

**Markdown示例：**
```markdown
- 一级项目
  - 二级项目
    - 三级项目
```

**效果：**
- 一级项目
  - 二级项目
    - 三级项目

### 五、引用与强调

#### 块引用（Blockquote）
- **背景色**：浅灰色（#F5F5F5）或无
- **左边框**：2-3 pt 深灰色（#CCCCCC）
- **左缩进**：1.0 cm
- **字体**：宋体，11 pt，斜体（可选）
- **上下边距**：6 pt
- **段落间距**：3 pt

**Markdown示例：**
```markdown
> 这是一个块引用
> 可以包含多行内容
> 用于引用诗歌或重要观点
```

**效果：**
> 这是一个块引用
> 可以包含多行内容
> 用于引用诗歌或重要观点

#### 加粗（Bold）
- **格式**：**文本** 或 __文本__
- **字体**：加粗
- **用途**：强调核心概念、专有名词、关键观点

**Markdown示例：**
```markdown
**这是加粗的文本**
```

**效果：**
**这是加粗的文本**

#### 斜体（Italic）
- **格式**：*文本* 或 _文本_
- **字体**：斜体
- **用途**：书名、外语词汇、强调

**Markdown示例：**
```markdown
*这是斜体的文本*
```

**效果：**
*这是斜体的文本*

#### 加粗斜体
- **格式**：***文本*** 或 ___文本___
- **用途**：最高级强调

**Markdown示例：**
```markdown
***这是加粗斜体的文本***
```

**效果：**
***这是加粗斜体的文本***

### 六、代码与代码块

#### 行内代码
- **格式**：`代码`
- **背景色**：浅灰色（#F0F0F0）
- **字体**：Courier New 或 Monaco，10 pt
- **内边距**：2 pt

**Markdown示例：**
```markdown
这是一个`行内代码`示例
```

**效果：**
这是一个`行内代码`示例

#### 代码块
- **背景色**：深灰色（#2D2D2D）或浅灰色（#F5F5F5）
- **字体**：Courier New，10 pt
- **文字颜色**：白色（深背景）或黑色（浅背景）
- **边框**：1 pt 灰色
- **内边距**：10 pt
- **圆角**：4 pt（可选）

**Markdown示例：**
````markdown
```python
def hello():
    print("Hello, World!")
```
````

**效果：**
```python
def hello():
    print("Hello, World!")
```

### 七、分隔线

- **格式**：--- 或 *** 或 ___
- **颜色**：浅灰色（#CCCCCC）
- **粗度**：1 pt
- **上下边距**：12 pt

**Markdown示例：**
```markdown
---
```

**效果：**

---

### 八、表格格式

#### 表格结构
- **边框**：1 pt 灰色（#CCCCCC）
- **表头背景**：浅蓝色（#E8F4F8）或浅灰色（#F0F0F0）
- **表头字体**：加粗，12 pt
- **单元格字体**：正常，11 pt
- **单元格内边距**：8 pt
- **行高**：1.5倍

#### 表格对齐
- **表头**：居中对齐
- **数据**：根据内容类型对齐（数字右对齐，文本左对齐）

**Markdown示例：**
```markdown
| 列1 | 列2 | 列3 |
|-----|-----|-----|
| 数据1 | 数据2 | 数据3 |
| 数据4 | 数据5 | 数据6 |
```

**效果：**
| 列1 | 列2 | 列3 |
|-----|-----|-----|
| 数据1 | 数据2 | 数据3 |
| 数据4 | 数据5 | 数据6 |

### 九、链接与脚注

#### 超链接
- **格式**：[链接文本](URL)
- **颜色**：蓝色（#0066CC）
- **下划线**：有
- **鼠标悬停**：颜色变深（#003399）

**Markdown示例：**
```markdown
[这是一个链接](https://example.com)
```

**效果：**
[这是一个链接](https://example.com)

#### 参考式链接
- **格式**：[链接文本][参考]
- **参考定义**：[参考]: URL

**Markdown示例：**
```markdown
这是一个[参考链接][1]

[1]: https://example.com
```

#### 脚注
- **格式**：文本[^1]
- **脚注定义**：[^1]: 脚注内容
- **脚注字号**：10 pt
- **脚注颜色**：灰色（#666666）

**Markdown示例：**
```markdown
这是一个脚注示例[^1]

[^1]: 这是脚注的内容
```

### 十、图片与媒体

#### 图片插入
- **格式**：![alt文本](图片路径)
- **最大宽度**：页面宽度的90%
- **对齐**：居中
- **边框**：1 pt 浅灰色（可选）
- **图片下方**：添加图片标题（可选）

**Markdown示例：**
```markdown
![图片描述](./image.png)
```

#### 图片标题
- **格式**：在图片下方添加
- **字号**：10 pt
- **颜色**：灰色（#666666）
- **对齐**：居中
- **格式**：*图1：图片描述* 或 **图1：图片描述**

**Markdown示例：**
```markdown
![示例图片](./image.png)

*图1：这是图片的标题*
```

## PDF 输出格式规范

### 一、页面设置

#### 页面尺寸与方向
- **纸张**：A4（210 × 297 mm）
- **方向**：纵向（Portrait）
- **缩放**：100%

#### 页边距
```
上：2.5 cm
下：2.5 cm
左：2.5 cm
右：2.5 cm
```

#### 页眉与页脚
- **页眉**：
  - 内容：课程名称 或 空白
  - 距顶部：1.25 cm
  - 字号：10 pt
  - 颜色：灰色（#666666）

- **页脚**：
  - 内容：页码（右对齐）
  - 距底部：1.25 cm
  - 字号：10 pt
  - 颜色：灰色（#666666）
  - 格式：第 X 页，共 Y 页（可选）

### 二、字体设置

#### 系统字体映射
| Markdown | PDF输出 | 备选方案 |
|----------|--------|--------|
| 正常 | 宋体 12pt | Calibri 12pt |
| 加粗 | 黑体 12pt | Arial Bold 12pt |
| 斜体 | 宋体斜体 12pt | Calibri Italic 12pt |
| 代码 | Courier New 10pt | Monaco 10pt |

#### 字体嵌入
- **嵌入所有字体**：是
- **子集嵌入**：是（仅包含使用的字符）

### 三、颜色设置

#### 标准颜色
| 用途 | 颜色代码 | RGB值 |
|------|--------|-------|
| 正文 | #000000 | 0, 0, 0 |
| 标题 | #000000 | 0, 0, 0 |
| 链接 | #0066CC | 0, 102, 204 |
| 引用背景 | #F5F5F5 | 245, 245, 245 |
| 表格边框 | #CCCCCC | 204, 204, 204 |
| 表头背景 | #E8F4F8 | 232, 244, 248 |
| 脚注 | #666666 | 102, 102, 102 |

### 四、行距与段距

#### 行距
- **正文**：1.5倍行距
- **标题**：1.2倍行距
- **代码块**：单倍行距
- **表格**：1.5倍行距

#### 段距
- **段前**：0 pt
- **段后**：6 pt
- **标题后**：12 pt
- **列表项间**：3 pt

### 五、分页设置

#### 分页规则
- **标题分页**：一级标题前自动分页
- **孤行控制**：启用（避免标题在页末）
- **段落分页**：禁用段落分页

#### 页码设置
- **起始页码**：1
- **格式**：阿拉伯数字
- **位置**：页脚右对齐
- **首页显示**：是

### 六、PDF元数据

#### 文档信息
- **标题**：课程笔记标题
- **作者**：Manus AI
- **主题**：课程笔记
- **关键词**：课程名称，相关主题
- **创建者**：Manus PDF Converter

#### 安全设置
- **打印**：允许
- **复制**：允许
- **修改**：禁止
- **密码**：无

## 完整格式示例

### Markdown 源文件结构

```markdown
# 课程笔记标题

## 第一章 章节名称

### 1.1 小节标题

　　这是一个缩进的段落。首行缩进两个全角空格。这是正文内容，字号为12pt，行距为1.5倍。

　　这是第二个段落，也有缩进。

#### 1.1.1 子小节标题

　　这是子小节下的内容。

### 1.2 另一个小节

　　这个小节包含**加粗的关键词**和*斜体的强调*。

> "这是一个块引用，用于引用诗歌或重要观点。"
> 
> 可以包含多行内容。

#### 列表示例

1. 有序列表项1
2. 有序列表项2
   - 嵌套无序列表项
   - 另一个嵌套项
3. 有序列表项3

#### 表格示例

| 列1 | 列2 | 列3 |
|-----|-----|-----|
| 数据1 | 数据2 | 数据3 |
| 数据4 | 数据5 | 数据6 |

---

## 第二章 另一个章节

### 2.1 新的小节

　　这是新章节的内容。

```

- 三级标题：14pt 黑体，上下各14pt和8pt间距
- 正文：12pt 宋体，1.5倍行距，首行缩进2字符

## 注意事项

1. **一致性**：全文保持格式一致
2. **可读性**：优先考虑阅读体验
3. **专业性**：符合学术文档标准
4. **兼容性**：确保PDF转换正确
5. **可访问性**：使用合理的颜色对比度
